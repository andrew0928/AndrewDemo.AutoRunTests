# Test Case: 啤酒數量 4（折扣兩次）

## Context

- shop: shop123  
- user: { user: andrew, password: 123456 }  
- location: { id: zh-TW, time-zone: UTC+8, currency: TWD }  

## Given

- 測試前請清空購物車  
- 指定測試商品: 啤酒

## When

1. 使用者登入後，建立一個新的購物車。  
2. 將 **4 件**啤酒放入購物車（可一次或分批加入），確保購物車中啤酒總數量為 4。  
3. 對購物車執行金額試算，以取得應付總額與折扣資訊。  

## Then 

- 購物車僅含啤酒，數量顯示為 **4**。  
- 折扣應在第 2、4 件各套用一次「六折」：  
  - 總折扣金額 = `0.4 × 單價 × 2`。  
- **應付總額** = `4 × 單價 − 0.8 × 單價 = 3.2 × 單價`。
